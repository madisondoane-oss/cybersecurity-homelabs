# System Metrics Analytics Lab

## Overview
This project demonstrates a lightweight analytics workflow using simulated system telemetry data.
The goal is to analyze CPU, memory, and disk I/O metrics over time, detect anomalies, and extract operational insights.

## Tools Used
- Python
- pandas
- numpy
- matplotlib
- SQLite
- Jupyter Notebook

## Workflow

1. Generated simulated system telemetry data (CPU, memory, disk I/O)
2. Performed exploratory data analysis with pandas
3. Implemented statistical anomaly detection using mean + 2*standard deviation
4. Visualized anomalies using matplotlib
5. Stored the dataset in SQLite and queried high-CPU events with SQL

## Skills Demonstrated
- Python data analysis
- Time-series monitoring
- Statistical anomaly detection
- SQL querying
- Data visualization

## Purpose
This lab sumulates the type of ananlytics workflow used in system monitoring and operational analytics environments.
